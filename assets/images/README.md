# Portfolio Website - Assets

## Images Folder

This folder should contain the following images for your portfolio website:

### Required Images:

1. **profile.jpg** - Your professional profile photo for the About section
   - Recommended size: 800x1000px
   - Format: JPG or PNG

2. **blog-1.jpg** - Featured image for the first blog post
   - Recommended size: 600x400px
   - Theme: Design-related

3. **blog-2.jpg** - Featured image for the second blog post
   - Recommended size: 600x400px
   - Theme: Business-related

4. **blog-3.jpg** - Featured image for the third blog post
   - Recommended size: 600x400px
   - Theme: Tips/Tutorial-related

### Image Sources:

You can get free high-quality images from:
- **Unsplash** (https://unsplash.com)
- **Pexels** (https://pexels.com)
- **Pixabay** (https://pixabay.com)

### Tips:

- All images should be optimized for web (under 500KB each)
- Use consistent color tones across images
- Consider using a professional headshot for your profile
- For blog images, choose images that relate to the blog post topics

## Placeholder

Until you add your images, the website will show placeholder areas. The design is fully responsive and will work with any image sizes, but the recommended dimensions will give the best results.
